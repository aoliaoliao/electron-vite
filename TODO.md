# 读取本地文件夹

## 交互步骤

1. 在菜单栏中设置 open folder 菜单，选择之后打开文件弹框，选择一个文件夹
2. 读取选择的文件夹结构，展示在两侧布局中的左侧
3. 单击文件夹中的文件，两侧布局中的右侧进入编辑+预览模式
4. 双击文件夹中的子文件夹，进入下一层目录
5. 在当前文件夹中空白处右击，可以新建文件，上传媒体资源
6. 单机选中某个文件或资源，右击出现菜单可以删除
7. 右侧顶部设置pubish，Draft，Delete等按钮。
